# *본격 알고리즘 통달 스터디*

*통달 - 사물의 이치나 지식, 기술 따위를 훤히 알거나 아주 능란하게 하다.*

- *프로그래밍 언어 JavaScript를 활용하여*
- *기초부터 시작하여 성장하는* `스터디`
- *혼자만의 성장이 아닌*
- *함께 성장하는 것을 목표로 하는 스터디*

<br/>

## PR 작성 규칙
- 하나의 문제마다 하나의 브랜치를 만들어서 PR을 작성한다.
- 브랜치 이름은 아래의 같은 규칙을 따른다.
- 문제번호는 노션에 적혀있는 문제의 번호를 따른다.

```
아이디/몇과_문제번호

tiaz0128/1_1
```
